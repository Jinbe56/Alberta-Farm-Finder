Solarpunk aesthetic — organic, hopeful, community-rooted

Design Philosophy

Inspired by Luke Humphris' When Society Collapsed series and the solarpunk movement. This isn't a corporate marketplace — it's a community tool that feels like walking into a warm, sun-lit farm stand where someone hand-lettered the signs.

Key principles: Soft, rounded edges everywhere (nothing sharp or corporate). Earthy palette that feels grown, not designed. Serif headings for warmth and personality. Generous whitespace — let things breathe like an open field. Every interaction should feel unhurried and human.

Design Rules

Corners: Always rounded. Cards get 20px, buttons get pill (999px), inputs get 8px. Nothing sharp.
Shadows: Warm-tinted (brown-based rgba), never cold gray. Subtle by default, more pronounced on hover.
Backgrounds: Cream (#FDFAF3), never pure white (#FFF). Cards can be white but the page never is.
Borders: Soft, warm gray (#E5DDD0). Never stark or dark.
Spacing: Generous. Let everything breathe. When in doubt, add more whitespace.
Animations: Gentle, slow transitions (0.2-0.3s ease). Cards lift slightly on hover. Nothing bounces or flashes.
Icons: If used, line-style only (not filled). Consider hand-drawn style icons to match the organic feel.
Photos: Farm photos should be warm-toned. Consider a subtle warm filter overlay to keep visual consistency.
Voice: Friendly, plain language. "Find what's growing near you" not "Discover local agricultural products." Write like you're talking to a neighbour.
